Java 命名规则

Aug 31st, 2015 4:19 pm

约定俗成命名规则

  • 类名的首字母大写；
  • 变量名和方法名的首字母小写；
  • 运用驼峰标志（如果名字由好几个单词构成，除了首字母遵循类名、变量名、方法名的命名规则外，其它单词的首字母都要大写。
